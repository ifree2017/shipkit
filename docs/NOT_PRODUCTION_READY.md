# Not Production-Ready

ShipKit is currently alpha-stage software.

It is suitable for:

- internal pilots
- SDLC harness design research
- project scaffolding experiments
- early agent workflow evaluation
- structured documentation and gate experimentation

It is not yet suitable for:

- fully automated production delivery
- unsupervised production deployment
- regulated or high-compliance delivery without human review
- replacing human PM, architecture, QA, security, or release approval

## Required Human Approval

The following must remain human-approved:

- customer-facing commitments
- quotes, contracts, and scope boundaries
- production releases
- data migrations
- destructive operations
- third-party integration activation
- security-sensitive changes
- client-facing delivery packages

## Alpha Rule

Use ShipKit to structure and verify work. Do not use it as an autonomous delivery authority.
