# Changelog

All notable changes to ShipKit will be documented in this file.

## 0.2.0-alpha.0 - Unreleased

### Added

- Alpha release-readiness documentation.
- Not-production-ready status note.
- Public differentiation statement for ShipKit as an SDLC delivery control layer.
- Release notes for `v0.2.0-alpha`.
- Expanded example project skeletons for `simple-solo` and `standard-team` profiles.
- Issue templates for bugs, feature requests, and integration candidates.
- Release readiness gate and CI workflow.

### Changed

- Updated README positioning around alpha status, roadmap, and examples.
- Expanded npm keywords for SDLC agent harness discovery.

### Known Limitations

- ShipKit is alpha-stage software and not production-ready.
- Runner adapters are mostly workspace/file adapters; long-running execution adapters are planned.
- Several skills and workflows remain scaffold-level.
