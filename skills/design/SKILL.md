# Design Layer

Create or update project-specific design artifacts:

- `design/DESIGN.md`
- `design/ui-spec.md`
- `design/components.md`
- `design/visual-review.md`

Rules:

- DESIGN.md must be project-specific.
- Do not copy third-party brand/design documents without review.
- UI specs must trace to PRD and acceptance criteria.
- Visual review must produce evidence before delivery.
- Accessibility, responsive behavior, spacing, state handling, and error states must be covered.
