# Data

Data work requires a data contract.

For each data issue or data contract, define:
1. Source
2. Destination
3. Field list
4. Field meaning
5. Format
6. Sample data
7. Validation rules
8. Error handling
9. Privacy and redaction rules
10. Acceptance evidence

Do not build data flows from undocumented assumptions.
