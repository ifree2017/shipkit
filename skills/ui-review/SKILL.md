# UI Review

Review UI output against:

- PRD
- DESIGN.md
- UI spec
- component conventions
- accessibility basics
- responsive behavior
- visual consistency
- customer-facing acceptance criteria

Output:

- pass/fail summary
- issues by severity
- screenshots or evidence references when available
- required fixes before delivery
