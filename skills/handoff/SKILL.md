# Handoff

Every stage must end with a handoff.

Required sections:
1. Stage
2. Inputs used
3. Outputs created
4. Confirmed items
5. Open questions
6. Risks
7. Decisions
8. Blockers
9. Next stage
10. Gate result

Gate result must be one of:
- PASS
- PASS_WITH_ASSUMPTIONS
- FAIL
- HUMAN_REVIEW_REQUIRED
