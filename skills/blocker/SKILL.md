# Blocker

Blockers are active stop signs, not passive notes.

For each blocker, produce:
1. Blocker summary
2. Current stage
3. Owner
4. Impact
5. Waiting on
6. Next action
7. Due date
8. Escalation path
9. Plan impact
10. Recommendation

Escalation guide:
- 24h: remind owner
- 48h: escalate to project owner
- 72h: recommend schedule or scope adjustment
