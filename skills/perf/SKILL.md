# Performance

Performance work must be measurable.

For each performance concern, define:
1. Scenario
2. Metric
3. Baseline
4. Target
5. Load profile
6. Measurement method
7. Bottleneck hypothesis
8. Optimization plan
9. Regression test
10. Acceptance threshold

Do not accept vague goals like "make it faster" without a measurable target.
