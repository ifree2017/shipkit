# Debt

Debt is allowed only when it is visible, classified, and accepted.

For each debt item, produce:
1. Debt type
2. Cause
3. Affected modules
4. Risk
5. Temporary behavior
6. Long-term fix
7. Test impact
8. Delivery impact
9. Repayment recommendation
10. Delivery allowance

Classify:
- D0: cannot remain before delivery
- D1: can deliver only with explicit risk note
- D2: schedule later
- D3: internal improvement
