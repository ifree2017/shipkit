# Visual Review

Check:

- layout consistency
- spacing and alignment
- typography
- color usage
- empty/loading/error states
- responsive behavior
- interactive affordances
- form validation states
- customer-facing polish

Write results to `design/visual-review.md` or the project evidence folder.
