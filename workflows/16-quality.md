# Workflow: Quality

Use before quote, PRD, architecture, release, and delivery gates.

## Steps

1. Grade each quality dimension A-E.
2. Attach evidence for each grade.
3. Mark unknowns as gaps.
4. Compare against stage thresholds.
5. Decide pass, fail, pass with assumptions, or human approval.

## Outputs

- control/quality-scorecard.md
- control/gate-result.md
