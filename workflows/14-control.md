# Workflow: Control

Use when the project deviates from expected state.

## Steps

1. Sense current state from artifacts and signals.
2. Compare against expected stage output and gate rules.
3. Classify deviation.
4. Create corrective action.
5. Check human approval policy.
6. Verify after correction.
7. Record feedback if repeated.

## Outputs

- control/control-report.md
- handoff/control.handoff.md
